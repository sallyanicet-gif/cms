# 🎮 Soft West - Alien Plushies Store

Une boutique en ligne moderne pour les plushies aliens avec détection d'humeur!

## 🚀 Déploiement sur GitHub Pages

Ce site est hébergé sur GitHub Pages et utilise Supabase comme backend.

### 📋 Fichiers inclus

- `index.html` - Page principale du site
- `app.js` - Logique JavaScript avec Supabase intégré ✅ **Clés configurées**
- `styles.css` - Styles CSS personnalisés

### ✨ Fonctionnalités

✅ Pages multiples (Accueil, Boutique, Contact, Admin)  
✅ Mode sombre/clair  
✅ Catalogue de produits dynamique  
✅ Dashboard admin avec CRUD  
✅ Authentification Supabase  
✅ Responsive design  
✅ Recherche et filtrage de produits

### 🌐 Accès au site

Une fois déployé sur GitHub Pages:
```
https://VOTRE_USERNAME.github.io/soft-west
```

Exemple: `https://neidjah.github.io/soft-west`

### 📱 Utilisation locale

```bash
# Installer un serveur local (Python)
python -m http.server 8000

# Ou avec Node.js
npx http-server
```

Puis ouvrez `http://localhost:8000` dans votre navigateur.

### 🔧 Modification des fichiers

1. Modifiez les fichiers localement
2. Commitez: `git commit -am "Description"`
3. Poussez: `git push`
4. GitHub Pages se mettra à jour automatiquement (2-3 minutes)
